# 🎧 Spotify Clone

A full-featured Spotify Clone built using React + Vite with authentication support, responsive UI, and modern design principles.

---

## 📌 Features

* 🔐 User Authentication (Login / Signup)
* 🎵 Spotify-inspired modern UI
* 📱 Fully responsive design
* ⚡ Fast performance with Vite
* 🎧 Music player interface (play/pause)
* 🧭 Sidebar navigation
* 🔍 Search functionality (if implemented)
* 💾 Persistent user session (if using local storage / backend)

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite
* **Authentication:** (Supabase)
* **Styling:** CSS / Tailwind CSS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Khushal-Narsaria/Spotify-Clone.git
cd Spotify-Clone
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file in root:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

---

### 4️⃣ Run the project

```bash
npm run dev
```

---

### 5️⃣ Open in browser

```
http://localhost:5173
```
